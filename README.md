# Exam Hub – Online Examination System

A web-based platform for conducting and managing online examinations.

## Features
- Online exam interface
- Question display and navigation
- User-friendly exam interaction

## Technologies
- React.js
- JavaScript
- HTML
- CSS

## Description
Exam Hub is an online examination system where users can attempt exams
through a web interface. The system allows structured navigation between
questions and provides a simple platform for managing examination tasks.
